配列作成

  • 配列は [] で囲む

    • var traficLights = {"赤","青","黄色"};
    console.log(traficLights[0]) → 赤 と表示される

  • 要素の数は 配列変数名.length で求める

    • console.log(traficLights.length) → 3 と表示される｡

  • 配列をループで回す

    • for(var i =0; i<配列変数名.length; i++){
         console.log(配列変数名[i]);
    }
  • 配列をループで回す

    • for(var i =0; i<配列変数名.length; i++){
         console.log(配列変数名[i]);
    }
  • 配列の要素をランダムに取得する

    • var drink = drinks[Math.floor(Math.random() * drinks.length)];
